# Пенсионный калькулятор

## Как открыть на iPhone (через Vercel — бесплатно)

### Шаг 1: Загрузи на GitHub
1. Зайди на [github.com](https://github.com) и создай аккаунт (если нет)
2. Нажми **New repository** → назови `pension-calculator` → Create
3. Загрузи все файлы из этой папки

### Шаг 2: Задеплой на Vercel
1. Зайди на [vercel.com](https://vercel.com)
2. Войди через GitHub
3. Нажми **Add New Project** → выбери репозиторий `pension-calculator`
4. Нажми **Deploy** — готово!
5. Получишь ссылку вида `pension-calculator.vercel.app`

### Шаг 3: Открой на iPhone
1. Открой ссылку в **Safari**
2. Нажми кнопку «Поделиться» (квадрат со стрелкой вверх)
3. Выбери **«На экран "Домой"»**
4. Приложение появится как иконка на рабочем столе 📱

## Локальный запуск (для разработки)
```bash
npm install
npm run dev
```
